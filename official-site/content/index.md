# 欢迎来到 Stello

**Stello** 是一个为中文用户设计的**静态网站生成器**。使用简洁的中文 DSL 和 Markdown，轻松构建结构清晰、可预测的静态网站。

## 核心特性

### 🌐 中文优先设计
- **中文 DSL** - 配置文件使用中文编写，更直观易懂
- **无隐式默认值** - 所有规则明确，缺失配置直接报错
- **明确的错误提示** - 帮助你快速定位和修复问题

### 🎨 灵活的页面结构
- **块系统** - 通过声明块列表 `[title, content, author]` 定义页面结构
- **显式排序** - 块的顺序即渲染顺序，完全可控
- 支持块类型：标题、正文、导航、作者信息、文章列表

### Markdown + DSL 混合格式
```markdown
标题 "我的文章"
块列表 [title, content]
日期 "2025-12-25"

# 文章正文从这里开始
```

### 🚀 自动导航生成
- 自动识别加入导航的页面
- 按 `排序` 字段自动排序
- 生成统一的导航菜单

### 🎭 开箱即用的主题
- 现代化的默认主题
- 响应式设计，适配各种屏幕
- 支持自定义 CSS
- 可创建新主题
