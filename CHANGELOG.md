# 更新日志

所有对此项目的显著更改都将记录在此文件中。

这个项目遵循 [语义版本控制](https://semver.org/) 规范。

## [1.0.0] - 2025-12-25

### 新增

- ✨ 完整的中文 DSL 实现
  - 自定义词法分析器，支持中文关键字
  - 递归下降语法分析器
  - 完整的验证和错误报告系统
  
- 📄 Markdown + DSL 混合格式
  - 在 Markdown 头部使用 DSL 配置
  - 自动提取和解析配置
  - 支持所有页面配置字段
  
- 🎨 灵活的块系统
  - 5 种块类型：title、content、nav、author、article_list
  - 显式块声明，块顺序即渲染顺序
  - 块类型验证
  
- 🔗 自动导航生成
  - 自动识别加入导航的页面
  - 按排序字段自动排序
  - 在每个页面自动生成导航菜单
  
- 🔍 完整的验证系统
  - URL 冲突检测
  - 缺失字段检查
  - 块类型验证
  - 日期格式验证
  - 详细的错误报告
  
- 🎭 主题系统
  - 默认响应式主题
  - 支持自定义 CSS
  - 支持自定义块模板
  - 支持创建新主题
  
- 📚 完整的文档
  - README.md - 项目介绍和快速开始
  - API.md - 完整的编程接口文档
  - PROJECT_SUMMARY.md - 项目架构说明
  - CONTRIBUTING.md - 贡献指南
  
- 🌐 官方网站
  - 首页和下载页
  - 自动导航生成
  - 源代码打包下载

### 改进

- 优化构建流程的可视化反馈
- 改进导航和列表排序逻辑
- 完善错误处理和报告机制
- 清晰的代码注释和文档字符串

### 技术细节

- Python 3.8+ 支持
- 无外部 DSL 解析库依赖
- 轻量级依赖（仅需 Jinja2 和 Markdown）
- 完全静态 HTML 输出
- 支持 UTF-8 编码
